Growpy Architecture

As follow the architecture design with two different component and four layers (html/javascript client, json rest api, controller python/django, ORMs [django-orm/SQLallchemy]):

Goals:

  • DB decouple
  • Independent RDBMS (all SQLAlchemy, django supported)
  • Data transport by SSH protocol
  • Deploy with gunicorn and nginx are sufficient.

Configure virtualenv .bash_profile vars and create virtualenv

mkvirtualenv webgrowpy
workon webgrowpy
cd $VIRTUAL_ENV
pip install django
pip install gunicorn
git clone https://github.com/mnothic/django-growpy.git
cd django-growpy
gunicorn webfront.wsgi

Then configure nginx proxy and static files server, in nginx.conf add this:

    server {
       listen       80;
       server_name  your_server.name;
       access_log   /var/log/nginx/nginx-access.log;

       location  /static/ {
       root /home/user/.virtualenvs/webgrowpy/django-growpy/growpy/templates;
    }

    location / {
        proxy_pass   http://127.0.0.1:8000;
  }
