Growpy Architecture

As follow the architecture design with two different component and four layers (html/javascript client, json rest api, controller python/django, ORMs [django-orm/SQLallchemy]): Goals:

  • db decoupled
  • independent RDBMS (all SQLAlchemy, django supported)
  • data transport by SSH protocol
  • deploy with gunicorn and nginx are sufficient.
